# CineScope – Movie Search Web Application

## CineScope is a web application that allows users to search and explore movies using real-time data from a public API. The application fetches movie information dynamically and displays it in an interactive and responsive user interface.Users can search for movies by title, view basic movie details such as posters, release year, and type, and interact with the results using features such as sorting, filtering, and favorites.

## The main objectives of this project are:

Integrate a public API to fetch real-time data
Display API data dynamically on a webpage
Implement HOF, features like search, filtering, and sorting
Build a responsive user interface using CSS.

## API Used-

This project uses the OMDb API (Open Movie Database API) to fetch movie data.
Website link - https://www.omdbapi.com/

The API provides movie data such as:

Movie Title
Release Year
Movie Poster
Movie Type
Additional movie details
